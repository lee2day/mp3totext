# filetotext

This program helps to convert .mp3 to .wav and the audio file to text. 

> Python 3.6

### Installation
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.
```sh
$ sudo pip3 install SpeechRecognition
$ sudo pip3 install pydub
```
## Usage

```python
python3 ./main.py
```
